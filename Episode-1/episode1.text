CDN: CDN stands for Content Delivery Network. The purpose of a CDN is to deliver web content, such as images, videos, CSS files, JavaScript files, and other static assets, to users more efficiently and quickly.

Use of CDN: Improved Performance, Scalability, Caching. 
Also, CDNs are used to enhance website and application performance, improve scalability, optimize content delivery, ensure global reach, provide security features, and offer valuable analytics and reporting capabilities.
