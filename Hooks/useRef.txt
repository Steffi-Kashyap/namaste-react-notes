🎣 useRef is a hook in React, a JavaScript library for building user interfaces.
 It 🚪 provides a way to access and interact with a DOM element or a React component directly.

🚀 Advantages of useRef

🔄 Avoiding Unnecessary Renders
🤝 Direct Interaction with DOM
📌 Persistent Values
🚫 Disadvantages of useRef

🚨 Not Triggering Re-renders
🔄 Potential for Mutable State
